# Zeon---desktop-aplikacija
Zeon Desktop  je WPF stolna aplikacija u C#/.NET-u za fitness trenere. Građena na troslojnoj arhitekturi uz Entity Framework, aplikacija nudi napredno upravljanje klijentima, planovima prehrane i LINQ analitiku povratnih informacija, uz automatsko generiranje i izvoz grafičkih izvještaja o napretku u PDF formatu
